sensed
======

A simple 2D node editor. Experimenting with wireless sensor network editing/visualization. Just a quick timewaste..


prerequisites
=============

pip install PyOpenGL
pip install pysdl2
pip install cython ?

    SET VS90COMNTOOLS=%VS110COMNTOOLS%
    pip install cython
